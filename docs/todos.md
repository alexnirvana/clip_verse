## 注意事项

1. **时区统一**: 所有时间统一使用中国时区(Asia/Shanghai, UTC+8)
2. **敏感数据**: 密码、私钥等敏感内容默认加密或启用安全模式
3. **性能考虑**: 大图片(>5MB)默认不保存，可配置
4. **存储策略**: SQLite 存储元数据和文本引用，文件单独存储
5. **错误恢复**: 实现数据库损坏恢复机制
6. **兼容性**: 确保跨平台兼容(Windows/macOS/Linux)
